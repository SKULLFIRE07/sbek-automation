SBEK Automation — Setup Checklist

All credentials and API keys needed to run the system. Section-by-section guide.

1. WooCommerce Store

Connects your online store to sync orders, products, and customers automatically.

FieldStatusHow to Get
Store URL (Required) Have (ENV) Your WooCommerce store URL, e.g. https://store.sbek.com
Consumer Key (Required) Have (ENV) 1. Go to your WooCommerce admin panel 2. WooCommerce → Settings → Advanced → REST API 3. Click "Add Key" → Permissions: Read/Write → Generate 4. Copy the Consumer Key (starts with ck_)
Consumer Secret (Required) Have (ENV) Generated alongside the Consumer Key above (starts with cs_). Copy it immediately — it's shown only once.
Webhook Secret (Required) Have (ENV) 1. WooCommerce → Settings → Advanced → Webhooks 2. Click "Add Webhook" 3. Copy the Secret shown at the top

2. Google Sheets & Drive

Syncs order data to Google Sheets and uploads product creatives to Google Drive.

FieldStatusHow to Get
Google Login (Required) Need to Connect 1. Open the SBEK Dashboard → Settings 2. In "Google Sheets & Drive" section, click "Login with Google" 3. Sign in with your Gmail account and allow access 4. You'll be redirected back. It will show "Connected as your@gmail.com"
Google Sheet URL (Required) Need to Create 1. Go to https://sheets.google.com 2. Click "+" to create a new blank spreadsheet 3. Name it "SBEK Orders" (or anything you like) 4. Copy the full URL from the browser address bar 5. Paste it in the dashboard Settings → "Google Sheet URL" field 6. Click "Setup Sheet & Drive" — the system will auto-create all tabs
Google Drive Folder URL Optional Leave empty — the system auto-creates an "SBEK Creatives" folder. Or: Go to https://drive.google.com → Create a folder → paste its URL.
Note: The "Login with Google" button requires OAuth credentials (Client ID & Secret) to be pre-configured by the developer in Railway environment variables. If the button doesn't work, ask the developer to set GOOGLE_OAUTH_CLIENT_ID and GOOGLE_OAUTH_CLIENT_SECRET.

3. WhatsApp (Meta Cloud API)

Sends order confirmations, shipping updates, and review requests via WhatsApp.

FieldStatusHow to Get
Phone Number ID (Required) Have (ENV) 1. Go to https://business.facebook.com 2. Navigate to WhatsApp → API Setup 3. Copy the "Phone Number ID" (numeric, e.g. 123456789012345)
Permanent Access Token (Required) Have (ENV) 1. Go to https://business.facebook.com 2. WhatsApp → API Setup → scroll down 3. Generate a Permanent Access Token (not the temporary one) 4. Copy the token

4. Email (SMTP)

Sends order confirmation emails, shipping notifications, and marketing emails.

FieldStatusHow to Get
SMTP Host (Required) Have (ENV) Gmail: smtp.gmail.com
Outlook: smtp.office365.com
SMTP Port (Required) Have (ENV) 587 (for Gmail and most providers)
Email Address (Required) Have (ENV) Your full Gmail address, e.g. orders@sbek.com
App Password (Required) Have (ENV) For Gmail (must use App Password, NOT regular password): 1. Go to https://myaccount.google.com/security 2. Enable 2-Step Verification (if not already on) 3. Go to https://myaccount.google.com/apppasswords 4. App name: type "SBEK" → click Create 5. Copy the 16-character password (e.g. abcd efgh ijkl mnop) 6. Paste it as the SMTP Password (remove spaces)
From Display Name Optional Format: SBEK <orders@sbek.com>

5. AI — Text & Image Generation

Powers product descriptions, SEO content, social captions, competitor analysis, and product image generation.

FieldStatusHow to Get
OpenRouter API Key (Required for text) Need 1. Go to https://openrouter.ai 2. Sign up / Log in 3. Dashboard → API Keys → Create Key 4. Copy the key (starts with sk-or-) 5. Add credits: Dashboard → Credits → Add funds ($5-10 to start)
Gemini API Key Optional (for images) 1. Go to https://aistudio.google.com/apikey 2. Click "Create API Key" 3. Select or create a Google Cloud project 4. Copy the key
Note: If you only set the Gemini key (no OpenRouter), it will be used for both text and images. OpenRouter gives better text quality and model choice.

6. Social Media (Postiz)

Auto-schedules and publishes posts to Instagram, Facebook, and Pinterest.

FieldStatusHow to Get
Postiz API Key (Required) Need 1. Go to https://app.postiz.com 2. Sign up / Log in 3. Settings → API → Generate Key 4. Copy the key
Postiz Base URL Optional Default: https://app.postiz.com/api/v1 — only change if self-hosting.

7. Brand Identity

Used across all emails, WhatsApp messages, AI-generated content, and social posts.

FieldStatusHow to Get
Brand Name (Required) Have (ENV) Your brand name, e.g. SBEK
Primary Color Optional Hex color code, e.g. #B8860B (used in email templates)
Website URL Optional e.g. https://sbek.com
Support Phone Optional e.g. +91XXXXXXXXXX
Support Email Optional e.g. support@sbek.com
Review Link Optional Google Business review URL or Trustpilot link
